# Gereksinimler

<Callout type={"info"}>Bu sayfada PowerNukkitX için eklenti geliştirmek için ihtiyacınız olan her şeyi bulabilirsiniz.</Callout>

İstediğiniz bir eklentiyi geliştirmek için şunlara ihtiyacınız olacak:
- Java Development Kit (JDK) 21 veya daha üstü
- Java programlama konusunda temel bilgi
- Bir PowerNukkitX sunucusu
- Bir IDE (Entegre Geliştirme Ortamı)
