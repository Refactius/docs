# Installing PowerNukkitX with the CLI (Recommended)

<Callout type={"warning"}>The new CLI is still in beta and may not function correctly. If you encounter issues, you can report a problem on the <a href={""}>GitHub Issue</a> page and use the <a href={""}>manual installation</a> method.</Callout>

## Prerequisites
- Make sure you meet the <a href="/requirements">requirements</a>
- Node.js 20 [Download](https://nodejs.org/en/download/)

## Usage
In this guide, we will use the name `server` for the server folder, but you can use any name you prefer.
<Steps>
    ### Create a folder
    <Tabs items={['linux/debian/windows']}>
        <Tabs.Tab>
            ```bash
            mkdir server
            cd server
            ```
        </Tabs.Tab>
    </Tabs>
    <FileTree>
        <FileTree.Folder name="server" defaultOpen children="" />
    </FileTree>

    ### Run the CLI
    <Callout type={"error"}>Ensure that the command is executed in the directory you just created, otherwise the CLI will not work correctly.</Callout>
    ```bash
    npx pnx-cli
    ```
    Follow the instructions to install your server more easily.
</Steps>
