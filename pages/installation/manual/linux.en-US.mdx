# Manual Installation of PowerNukkitX for Linux/Debian

The following steps are for Debian, but they can be adapted for other Linux distributions.

<Callout type="default">You will need `wget` or `curl` and `unzip` to download and extract the ZIP archive. If you don't have them, you can install them with `apt install wget unzip`.</Callout>

<Steps>
    ### Create a Folder for PowerNukkitX
    Create a folder for PowerNukkitX, for example, `server`:
    ```bash
    mkdir server
    ```

    ### Navigate to the Folder
    Navigate to the folder you created:
    ```bash
    cd server
    ```

    ### Download the ZIP Archive
    Download the `powernukkitx-run.zip` archive with the following command:
    <Tabs items={['wget', 'curl']}>
        <Tabs.Tab label="wget">
            ```bash
            wget https://github.com/PowerNukkitX/PowerNukkitX/releases/download/snapshot/powernukkitx-run.zip
            ```
        </Tabs.Tab>
        <Tabs.Tab label="curl">
            ```bash
            curl -O https://github.com/PowerNukkitX/PowerNukkitX/releases/download/snapshot/powernukkitx-run.zip
            ```
        </Tabs.Tab>
    </Tabs>

    ### Extract the Archive
    Extract the contents of the ZIP archive into the folder you created:
    ```bash
    unzip powernukkitx-run.zip
    ```

    The contents should look like this:
    <FileTree>
        <FileTree.Folder name="server" defaultOpen>
            <FileTree.Folder name="libs">
                <FileTree.File name="annotations-24.1.0.jar" />
                <FileTree.File name="asm-9.7.jar" />
                <FileTree.File name="checker-qual-3.41.0.jar" />
                <FileTree.File name="collections-23.1.3.jar" />
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.File name="powernukkitx-run.zip" />
            <FileTree.File name="cli.jar" />
            <FileTree.File name="powernukkitx.jar" />
        </FileTree.Folder>
    </FileTree>

    <Callout type="default">You can delete `cli.jar`, it will not be used because we will use `start.sh`.</Callout>
    ```bash
    rm cli.jar
    ```

    <Callout type="info">You can delete the ZIP archive after extraction.</Callout>
    ```bash
    rm powernukkitx-run.zip
    ```

    ### Download the start.sh File
    Download the `start.sh` startup file with this command:

    <Tabs items={['wget', 'curl']}>
        <Tabs.Tab label="wget">
            ```bash
            wget https://raw.githubusercontent.com/PowerNukkitX/scripts/master/start.sh
            ```
        </Tabs.Tab>
        <Tabs.Tab label="curl">
            ```bash
            curl -O https://raw.githubusercontent.com/PowerNukkitX/scripts/master/start.sh
            ```
        </Tabs.Tab>
    </Tabs>

    ### Grant Execution Permissions
    Grant execution permissions to the startup file:
    ```bash
    chmod +x start.sh
    ```

    ### Folder Structure
    Your folder structure should look like this:
    <FileTree>
        <FileTree.Folder name="server" defaultOpen>
            <FileTree.Folder name="libs">
                <FileTree.File name="annotations-24.1.0.jar" />
                <FileTree.File name="asm-9.7.jar" />
                <FileTree.File name="checker-qual-3.41.0.jar" />
                <FileTree.File name="collections-23.1.3.jar" />
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.File name="start.sh" />
            <FileTree.File name="powernukkitx.jar" />
        </FileTree.Folder>
    </FileTree>

    ### Start the Server
    Execute the startup file:
    ```bash
    ./start.sh
    ```

    After starting, your folder structure should look like this:

    <FileTree>
        <FileTree.Folder name="server" defaultOpen>
            <FileTree.Folder name="command_data">
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.Folder name="libs">
                <FileTree.File name="annotations-24.1.0.jar" />
                <FileTree.File name="asm-9.7.jar" />
                <FileTree.File name="checker-qual-3.41.0.jar" />
                <FileTree.File name="collections-23.1.3.jar" />
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.Folder name="logs">
                <FileTree.File name="server.log" />
            </FileTree.Folder>
            <FileTree.Folder name="players">
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.Folder name="plugins">
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.Folder name="worlds">
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.File name="start.sh" />
            <FileTree.File name="powernukkitx.jar" />
        </FileTree.Folder>
    </FileTree>

    ### Your Server is Now Online
    You can now connect to your server using your server's IP address and the default port `19132`.
</Steps>
