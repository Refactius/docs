# Installation de PowerNukkitX en manuel pour Linux/Debian

Les étapes suivantes sont pour Debian, mais elles peuvent être adaptées pour d'autres distributions Linux.

<Callout type="default">Il nous faudra `wget` ou `curl` et `unzip` pour télécharger l'archive ZIP et l'extraire. Si vous ne les avez pas, vous pouvez les installer avec `apt install wget unzip`.</Callout>

<Steps>
    ### Créez un dossier pour PowerNukkitX
    Créez un dossier pour PowerNukkitX, par exemple `server` :
    ```bash
    mkdir server
    ```

    ### Accédez au dossier
    Accédez au dossier que vous avez créé :
    ```bash
    cd server
    ```

    ### Téléchargez l'archive ZIP
    Téléchargez l'archive ZIP `powernukkitx-run.zip` avec la commande suivante :
    <Tabs items={['wget', 'curl']}>
        <Tabs.Tab label="wget">
            ```bash
            wget https://github.com/PowerNukkitX/PowerNukkitX/releases/download/snapshot/powernukkitx-run.zip
            ```
        </Tabs.Tab>
        <Tabs.Tab label="curl">
            ```bash
            curl -O https://github.com/PowerNukkitX/PowerNukkitX/releases/download/snapshot/powernukkitx-run.zip
            ```
        </Tabs.Tab>
    </Tabs>

    ### Extrayez l'archive
    Extrayez le contenu de l'archive ZIP dans le dossier que vous avez créé :
    ```bash
    unzip powernukkitx-run.zip
    ```

    Le contenu devrait ressembler à ceci :
    <FileTree>
        <FileTree.Folder name="server" defaultOpen>
            <FileTree.Folder name="libs">
                <FileTree.File name="annotations-24.1.0.jar" />
                <FileTree.File name="asm-9.7.jar" />
                <FileTree.File name="checker-qual-3.41.0.jar" />
                <FileTree.File name="collections-23.1.3.jar" />
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.File name="powernukkitx-run.zip" />
            <FileTree.File name="cli.jar" />
            <FileTree.File name="powernukkitx.jar" />
        </FileTree.Folder>
    </FileTree>

    <Callout type="default">Vous pouvez supprimer `cli.jar`, il ne sera pas utilisé car nous utiliserons `start.sh`.</Callout>
    ```bash
    rm cli.jar
    ```

    <Callout type="info">Vous pouvez supprimer l'archive ZIP après l'extraction.</Callout>
    ```bash
    rm powernukkitx-run.zip
    ```

    ### Téléchargez le fichier start.sh
    Téléchargez le fichier de démarrage `start.sh` avec cette commande :

    <Tabs items={['wget', 'curl']}>
        <Tabs.Tab label="wget">
            ```bash
            wget https://raw.githubusercontent.com/PowerNukkitX/scripts/master/start.sh
            ```
        </Tabs.Tab>
        <Tabs.Tab label="curl">
            ```bash
            curl -O https://raw.githubusercontent.com/PowerNukkitX/scripts/master/start.sh
            ```
        </Tabs.Tab>
    </Tabs>

    ### Donnez les permissions d'exécution
    Donnez les permissions d'exécution au fichier de démarrage :
    ```bash
    chmod +x start.sh
    ```

    ### Structure du dossier
    Vous devriez avoir une structure de dossier comme celle-ci :
    <FileTree>
        <FileTree.Folder name="server" defaultOpen>
            <FileTree.Folder name="libs">
                <FileTree.File name="annotations-24.1.0.jar" />
                <FileTree.File name="asm-9.7.jar" />
                <FileTree.File name="checker-qual-3.41.0.jar" />
                <FileTree.File name="collections-23.1.3.jar" />
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.File name="start.sh" />
            <FileTree.File name="powernukkitx.jar" />
        </FileTree.Folder>
    </FileTree>

    ### Démarrez le serveur
    Exécutez le fichier de démarrage :
    ```bash
    ./start.sh
    ```

    Après le démarrage, vous devriez avoir une structure de dossier comme celle-ci :

    <FileTree>
        <FileTree.Folder name="server" defaultOpen>
            <FileTree.Folder name="command_data">
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.Folder name="libs">
                <FileTree.File name="annotations-24.1.0.jar" />
                <FileTree.File name="asm-9.7.jar" />
                <FileTree.File name="checker-qual-3.41.0.jar" />
                <FileTree.File name="collections-23.1.3.jar" />
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.Folder name="logs">
                <FileTree.File name="server.log" />
            </FileTree.Folder>
            <FileTree.Folder name="players">
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.Folder name="plugins">
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.Folder name="worlds">
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.File name="start.sh" />
            <FileTree.File name="powernukkitx.jar" />
        </FileTree.Folder>
    </FileTree>

    ### Votre serveur est maintenant en ligne
    Vous pouvez maintenant vous connecter à votre serveur en utilisant l'adresse IP de votre serveur et le port par défaut `19132`.
</Steps>
