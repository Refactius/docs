# PowerNukkitX’in Linux İçin Manuel Kurulumu

Aşağıdaki adımlar Debian içindir, ancak diğer Linux dağıtımları için de uyarlanabilir.

<Callout type="default">ZIP arşivini indirmek ve çıkarmak için `wget` veya `curl` ve `unzip` komutlarına ihtiyacınız olacak. Eğer yüklü değilse, `apt install wget unzip` ile yükleyebilirsiniz.</Callout>

<Steps>
    ### PowerNukkitX İçin Bir Klasör Oluşturun
    Örneğin `server` adında bir klasör oluşturun:
    ```bash
    mkdir server
    ```

    ### Klasöre Girin
    Oluşturduğunuz klasöre girin:
    ```bash
    cd server
    ```

    ### ZIP Arşivini İndirin
    `powernukkitx-run.zip` arşivini aşağıdaki komutlardan biriyle indirin:
    <Tabs items={['wget', 'curl']}>
        <Tabs.Tab label="wget">
            ```bash
            wget https://github.com/PowerNukkitX/PowerNukkitX/releases/download/snapshot/powernukkitx-run.zip
            ```
        </Tabs.Tab>
        <Tabs.Tab label="curl">
            ```bash
            curl -O https://github.com/PowerNukkitX/PowerNukkitX/releases/download/snapshot/powernukkitx-run.zip
            ```
        </Tabs.Tab>
    </Tabs>

    ### Arşivi Çıkartın
    ZIP arşivini oluşturduğunuz klasöre çıkarın:
    ```bash
    unzip powernukkitx-run.zip
    ```

    İçerik aşağıdaki gibi görünmelidir:
    <FileTree>
        <FileTree.Folder name="server" defaultOpen>
            <FileTree.Folder name="libs">
                <FileTree.File name="annotations-24.1.0.jar" />
                <FileTree.File name="asm-9.7.jar" />
                <FileTree.File name="checker-qual-3.41.0.jar" />
                <FileTree.File name="collections-23.1.3.jar" />
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.File name="powernukkitx-run.zip" />
            <FileTree.File name="cli.jar" />
            <FileTree.File name="powernukkitx.jar" />
        </FileTree.Folder>
    </FileTree>

    <Callout type="default">`cli.jar` dosyasını silebilirsiniz, çünkü `start.sh` dosyasını kullanacağız.</Callout>
    ```bash
    rm cli.jar
    ```

    <Callout type="info">ZIP arşivini çıkardıktan sonra silebilirsiniz.</Callout>
    ```bash
    rm powernukkitx-run.zip
    ```

    ### start.sh Dosyasını İndirin
    Başlangıç dosyasını şu komutla indirin:

    <Tabs items={['wget', 'curl']}>
        <Tabs.Tab label="wget">
            ```bash
            wget https://raw.githubusercontent.com/PowerNukkitX/scripts/master/start.sh
            ```
        </Tabs.Tab>
        <Tabs.Tab label="curl">
            ```bash
            curl -O https://raw.githubusercontent.com/PowerNukkitX/scripts/master/start.sh
            ```
        </Tabs.Tab>
    </Tabs>

    ### Çalıştırma İzinlerini Verin
    Başlangıç dosyasına çalıştırma izni verin:
    ```bash
    chmod +x start.sh
    ```

    ### Klasör Yapısı
    Klasör yapınız aşağıdaki gibi olmalıdır:
    <FileTree>
        <FileTree.Folder name="server" defaultOpen>
            <FileTree.Folder name="libs">
                <FileTree.File name="annotations-24.1.0.jar" />
                <FileTree.File name="asm-9.7.jar" />
                <FileTree.File name="checker-qual-3.41.0.jar" />
                <FileTree.File name="collections-23.1.3.jar" />
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.File name="start.sh" />
            <FileTree.File name="powernukkitx.jar" />
        </FileTree.Folder>
    </FileTree>

    ### Sunucuyu Başlatın
    Başlangıç dosyasını çalıştırın:
    ```bash
    ./start.sh
    ```

    Başlatıldıktan sonra klasör yapınız şöyle olmalıdır:

    <FileTree>
        <FileTree.Folder name="server" defaultOpen>
            <FileTree.Folder name="command_data">

            <FileTree.Folder name="logs">

            <FileTree.File name="start.sh" />
            <FileTree.File name="powernukkitx.jar" />
        </FileTree.Folder>
    </FileTree>

    ### Sunucunuz Artık Çevrimiçi
    Sunucunuzun IP adresi ve varsayılan port `19132` ile sunucuya bağlanabilirsiniz.
</Steps>
