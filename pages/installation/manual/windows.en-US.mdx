import { Callout, FileTree, Steps } from "nextra/components";

# Manual Installation of PowerNukkitX for Windows

The following steps are for Windows.
<Steps>
    ### Download the ZIP Archive
    Download the `powernukkitx-run.zip` archive from the PowerNukkitX <a href="">download page</a>.

    ### Create a Folder for PowerNukkitX
    Create a folder for PowerNukkitX, for example, `server`.

    ### Extract the Archive
    Extract the contents of the ZIP archive into the folder you created. It should look like this:
    <FileTree>
        <FileTree.Folder name="server" defaultOpen>
            <FileTree.Folder name="libs">
                <FileTree.File name="annotations-24.1.0.jar" />
                <FileTree.File name="asm-9.7.jar" />
                <FileTree.File name="checker-qual-3.41.0.jar" />
                <FileTree.File name="collections-23.1.3.jar" />
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.File name="powernukkitx-run.zip" />
            <FileTree.File name="cli.jar" />
            <FileTree.File name="powernukkitx.jar" />
        </FileTree.Folder>
    </FileTree>

    <Callout type="default">
        You can delete `cli.jar`, it will not be used since we will use `start.bat`.
    </Callout>

    <Callout type="info">
        You can delete the ZIP archive after extraction.
    </Callout>

    ### Download the start.bat File
    Download the `start.bat` startup file from the PowerNukkitX <a href="https://github.com/PowerNukkitX/scripts/tree/master">download page</a>.

    ### Folder Structure
    You should have a folder structure like this:

    <FileTree>
        <FileTree.Folder name="server" defaultOpen>
            <FileTree.Folder name="libs">
                <FileTree.File name="annotations-24.1.0.jar" />
                <FileTree.File name="asm-9.7.jar" />
                <FileTree.File name="checker-qual-3.41.0.jar" />
                <FileTree.File name="collections-23.1.3.jar" />
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.File name="start.bat" />
            <FileTree.File name="powernukkitx.jar" />
        </FileTree.Folder>
    </FileTree>

    ### Start the Server
    Double-click on `start.bat` to start the server.

    After starting, you should have a folder structure like this:

    <FileTree>
        <FileTree.Folder name="server" defaultOpen>
            <FileTree.Folder name="command_data">
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.Folder name="libs">
                <FileTree.File name="annotations-24.1.0.jar" />
                <FileTree.File name="asm-9.7.jar" />
                <FileTree.File name="checker-qual-3.41.0.jar" />
                <FileTree.File name="collections-23.1.3.jar" />
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.Folder name="logs">
                <FileTree.File name="server.log" />
            </FileTree.Folder>
            <FileTree.Folder name="players">
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.Folder name="plugins">
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.Folder name="worlds">
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.File name="start.bat" />
            <FileTree.File name="powernukkitx.jar" />
        </FileTree.Folder>
    </FileTree>

    ### Your Server is Online
    You can now connect to your server using your server's IP address and the default port `19132`.

    <Callout type="error">
        Warning: Some people may not be able to connect to their server using the local IP `127.0.0.1`. To fix this, a command is available.
    </Callout>
</Steps>
