# Installation de PowerNukkitX en manuel pour Windows

Les étapes suivantes sont pour Windows.
<Steps>
    ### Téléchargez l'archive ZIP
    Téléchargez l'archive ZIP `powernukkitx-run.zip` depuis la page de PowerNukkitX dans la <a href="">page de téléchargement</a>.

    ### Créez un dossier pour PowerNukkitX
    Créez un dossier pour PowerNukkitX, par exemple `server`.

    ### Extrayez l'archive
    Extrayez le contenu de l'archive ZIP dans le dossier que vous avez créé. Cela devrait ressembler à ceci :
    <FileTree>
        <FileTree.Folder name="server" defaultOpen>
            <FileTree.Folder name="libs">
                <FileTree.File name="annotations-24.1.0.jar" />
                <FileTree.File name="asm-9.7.jar" />
                <FileTree.File name="checker-qual-3.41.0.jar" />
                <FileTree.File name="collections-23.1.3.jar" />
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.File name="powernukkitx-run.zip" />
            <FileTree.File name="cli.jar" />
            <FileTree.File name="powernukkitx.jar" />
        </FileTree.Folder>
    </FileTree>

    <Callout type="default">Vous pouvez supprimer `cli.jar`, il ne sera pas utilisé car nous utiliserons `start.bat`.</Callout>

    <Callout type="info">Vous pouvez supprimer l'archive ZIP après l'extraction.</Callout>

    ### Téléchargez le fichier start.bat
    Téléchargez le fichier de démarrage `start.bat` depuis la page de PowerNukkitX dans la <a href="https://github.com/PowerNukkitX/scripts/tree/master">page de téléchargement</a>.

    ### Structure du dossier
    Vous devriez avoir une structure de dossier comme celle-ci :

    <FileTree>
        <FileTree.Folder name="server" defaultOpen>
            <FileTree.Folder name="libs">
                <FileTree.File name="annotations-24.1.0.jar" />
                <FileTree.File name="asm-9.7.jar" />
                <FileTree.File name="checker-qual-3.41.0.jar" />
                <FileTree.File name="collections-23.1.3.jar" />
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.File name="start.bat" />
            <FileTree.File name="powernukkitx.jar" />
        </FileTree.Folder>
    </FileTree>

    ### Démarrez le serveur
    Double-cliquez sur `start.bat` pour démarrer le serveur.

    Après le démarrage, vous devriez avoir une structure de dossier comme celle-ci :

    <FileTree>
        <FileTree.Folder name="server" defaultOpen>
            <FileTree.Folder name="command_data">
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.Folder name="libs">
                <FileTree.File name="annotations-24.1.0.jar" />
                <FileTree.File name="asm-9.7.jar" />
                <FileTree.File name="checker-qual-3.41.0.jar" />
                <FileTree.File name="collections-23.1.3.jar" />
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.Folder name="logs">
                <FileTree.File name="server.log" />
            </FileTree.Folder>
            <FileTree.Folder name="players">
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.Folder name="plugins">
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.Folder name="worlds">
                <FileTree.File name="..." />
            </FileTree.Folder>
            <FileTree.File name="start.bat" />
            <FileTree.File name="powernukkitx.jar" />
        </FileTree.Folder>
    </FileTree>

    ### Votre serveur est en ligne
    Vous pouvez maintenant vous connecter à votre serveur en utilisant l'adresse IP de votre serveur et le port par défaut `19132`.

    <Callout type="error">Attention : chez certaines personnes, il est impossible de se connecter à leur serveur avec l'IP locale `127.0.0.1`. Pour régler cela, une commande existe.</Callout>
</Steps>
